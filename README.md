# employees
# Employees SQL Project – README

## Overview
This project uses the `employees` database to perform HR and workforce analytics using SQL.

The project focuses on analyzing:
- Employees
- Departments
- Salaries
- Hiring trends
- Employee experience levels
- Salary rankings

---

# Main Objectives

- Analyze employee distribution
- Track hiring trends over years
- Calculate department average salaries
- Identify highest-paid employees
- Rank employees by salary within departments
- Segment employees based on experience level

---

# SQL Concepts Used

- SELECT Statements
- Aggregation Functions (`COUNT`, `AVG`, `MIN`, `MAX`)
- JOINs
- GROUP BY
- ORDER BY
- LIMIT
- CASE Statements
- Window Functions (`RANK`)
- Common Table Expressions (CTE)
- Date Functions
- Subqueries
